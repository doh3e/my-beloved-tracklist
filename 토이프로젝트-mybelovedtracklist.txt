토이 프로젝트

my beloved tracklist

- 테마 : 오늘의 내가 사랑한 음악
- 기능 : 회원 관리, 음악 등록, 등록한 음악 보기, 음악 캘린더 (spotify API 연동)


- 세부 기능

1. 회원 관리
- 회원가입 (가능하다면 카카오 로그인까지)
- 로그인/로그아웃
- 회원수정/탈퇴

받을 정보
아이디 비밀번호 닉네임 


2. 음악 등록
- 음악 검색 버튼 클릭 -> 새창에서 등록하고 싶은 음악을 검색 (API)
찾아서 선택하고 나면 원래 창으로 돌아감 -> input에 가져온 정보 입력되어있음
노래제목, 앨범, 가수, 앨범표지 보여짐
- 음악 등록 : 위의 기능 + 오늘 기분 + 메모
- DB 저장 시 : 내아이디, 노래제목, 앨범, 가수, spotify 가수uri, 앨범표지, 오늘기분, 메모


3. 등록한 음악 보기
- 등록한 음악 플레이 리스트 or CD모양 카드 형태로 보여주기
클릭 시 자세히 보기로 이동 -> 

(스포티파이 로그인 기능 구현 가능할 시)
- 플레이 리스트일 경우 멜론 같은 st의 플리 이미지 =>
클릭시 앨범표지 변경 & 1분 미리듣기

- 가수/제목 가나다 순 정렬 제공 / 기분별 가수별 필터 제공
- 수정 삭제 버튼


3-1. 상세보기
DB에 저장된 앨범 표지, 음악 제목, 앨범, 가수, 오늘기분, 메모 등의 정보 제공


4. 등록한 음악 수정/삭제
- 수정 : 수정 페이지 -> 등록과 동일한 폼 (API)
- 삭제 : 삭제 여부 확인창 -> 삭제


5. 뮤직 캘린더
한 달의 기분, 등록한 노래를 달력 형태로 보여주는 캘린더
클릭 시 상세보기로 이동
